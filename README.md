# Description
This repository presents the design and implementation of an 8-bit-Arithmetic-Logic-Unit(ALU) using Cadence Virtuoso with GPDK90(90nm CMOS Technology). 
The ALU is one of the core components of a processor and performs arithmetic and logical operations on 8-bit binary data.
The project include transistor level CMOS implementation, schematic and simulation of ALU.
# Feature
- 8 bit CMOS ALU Design
- Transistor level implentation
- Schematic Design
- Functional Simulation
# Operation Implemented
- 8 Bit Addition
- Bitwise AND
- Bitwise OR
- Bitwise XOR
# Tools used
- Cadence Virtuoso
- GPDK90(90nm CMOS Technology)
